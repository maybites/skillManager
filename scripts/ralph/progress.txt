## Codebase Patterns
- Typecheck: `uv run pyright src/` (pyright added as dev dep)
- Entry point: `uv run skillmanager` maps to `skillmanager.main:run`
- NiceGUI: run with `host="127.0.0.1"` for localhost-only
- Build system: hatchling, packages = ["src/skillmanager"]
- NiceGUI page: use `@ui.page("/")` inside `run()` for proper per-client isolation
- NiceGUI splitter: `ui.splitter(value=N)` where N is 0-100 (percentage), use `.before`/`.after` slots
- NiceGUI element click: use `element.on("click", lambda _e, x=x: handler(x))` to capture loop vars; add `# type: ignore[misc]` for pyright lambda type complaints
- NiceGUI state: use `dict[str, T | None]` containers for mutable state shared across closures (avoids unbound-var issues with pyright)
